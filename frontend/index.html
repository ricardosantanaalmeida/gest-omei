Gest-Omei

Exemplo de UI minimalista consumindo a API de gestão financeira.

Este front-end consome a API do backend rodando em http://localhost:5000. Ele envia o cabeçalho X-User-Role para simular perfis (Master/Admin/User/Viewer).
